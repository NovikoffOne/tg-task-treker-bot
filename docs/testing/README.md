## Быстрый старт

### Запуск всех тестов
```bash
cd task_tracker_bot
pytest tests/ -v
```

### Запуск конкретного теста
```bash
pytest tests/test_repositories.py -v
```

### Тестирование через MCP
См. [быстрое тестирование](quick-test-guide.md)

## Покрытие тестами

Текущее покрытие:
- ✅ База данных (100%)
- ✅ Репозитории (100%)
- ✅ Сервисы (основные функции)
- ✅ Валидаторы (100%)
- ✅ Синхронизация полей (100%)
- ✅ Интеграционные тесты (основные сценарии)

## Проверка покрытия

```bash
pip install pytest-cov
pytest task_tracker_bot/tests/ --cov=task_tracker_bot --cov-report=html
```
