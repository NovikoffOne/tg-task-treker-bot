## Версии проекта

### v2.0 (текущая)
Полнофункциональная система управления проектами с:
- Пространствами, досками, колонками
- Проектами с дашбордами
- Синхронизацией полей
- Приоритетами, метками, статистикой
- Визуализацией досок
- AI-агентами

**Спецификация:** [project-v2.md](project-v2.md)

### MVP 0.2
Расширенная версия с:
- Зависимостями досок
- Участниками проектов
- Дедлайнами задач
- Улучшенным просмотром задач

**Спецификация:** [mvp-0.2.md](mvp-0.2.md)

### v1.0 (устарела)
Базовая версия таск-трекера с простым CRUD для задач.

**Спецификация:** [deprecated/project-v1.md](deprecated/project-v1.md)

## Использование спецификаций

### Для разработчиков
1. Начните с [технической спецификации](technical.md) для понимания реализации
2. Изучите [спецификацию проекта v2](project-v2.md) для понимания функционала
3. Используйте [спецификацию MVP 0.2](mvp-0.2.md) при работе с новыми функциями

### Для тестировщиков
1. Используйте спецификации для создания тест-кейсов
2. Проверяйте соответствие реализации спецификациям
3. Отмечайте расхождения в багах

### Для пользователей
1. Изучите [руководство пользователя](../guides/user-guide.md) для практического использования
2. Используйте спецификации для понимания возможностей системы

## Структура спецификаций

- Критерии готовности
