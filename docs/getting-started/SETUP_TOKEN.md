# Как получить токен для Telegram бота

## Пошаговая инструкция

### Шаг 1: Откройте Telegram
Откройте приложение Telegram на вашем телефоне или компьютере.

### Шаг 2: Найдите BotFather
В поиске Telegram найдите **@BotFather** (это официальный бот от Telegram для создания ботов).

### Шаг 3: Начните диалог
Нажмите на @BotFather и отправьте команду:
```
/start
```

### Шаг 4: Создайте нового бота
Отправьте команду:
```
/newbot
```

### Шаг 5: Укажите имя бота
BotFather попросит вас ввести имя для вашего бота. Это имя будет отображаться в списке контактов.
Например: `My Task Tracker Bot`

### Шаг 6: Укажите username бота
BotFather попросит username для вашего бота. Username должен:
- Заканчиваться на `bot` (например: `mytasktracker_bot`)
- Быть уникальным (если имя занято, попробуйте другое)

### Шаг 7: Получите токен
После успешного создания бота, BotFather отправит вам сообщение с токеном, который выглядит примерно так:
```
123456789:ABCdefGHIjklMNOpqrsTUVwxyz
```

### Шаг 8: Сохраните токен
Скопируйте этот токен и вставьте его в файл `.env`:
```bash
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
```

## Важно!

⚠️ **НЕ ДЕЛИТЕСЬ ТОКЕНОМ** с другими людьми и не публикуйте его в открытых репозиториях!

Если токен попадет в чужие руки, они смогут управлять вашим ботом.

## Альтернативные команды BotFather

- `/mybots` - Управление вашими ботами
- `/token` - Получить токен существующего бота
- `/revoke` - Отозвать текущий токен и получить новый

## После получения токена

1. Откройте файл `task_tracker_bot/.env`
2. Замените `your_bot_token_here` на ваш реальный токен
3. Сохраните файл
4. Запустите бота: `python3 task_tracker_bot/bot.py`

## Проверка работы

После запуска бота с правильным токеном, вы должны увидеть в логах:
```
INFO - База данных инициализирована
INFO - Бот запущен и готов к работе!
```

Затем найдите вашего бота в Telegram по username и отправьте ему команду `/start`.
