## Быстрая установка

```bash
# 1. Клонируйте репозиторий
git clone <repository_url>
cd tg-task-treker-bot

# 2. Создайте виртуальное окружение
python -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate

# 3. Установите зависимости
pip install -r task_tracker_bot/requirements.txt

# 4. Настройте .env файл
cp .env.example .env
# Отредактируйте .env и добавьте BOT_TOKEN

# 5. Запустите бота
python task_tracker_bot/bot.py
```

## Первые команды

После запуска бота отправьте ему в Telegram:

1. `/start` - начать работу с ботом
2. `/help` - получить справку по командам
3. `/workspaces` - посмотреть список пространств
