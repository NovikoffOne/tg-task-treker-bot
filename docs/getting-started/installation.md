# Инструкция по запуску бота

## Способ 1: Использование скрипта (рекомендуется)

```bash
./start_bot.sh
```

## Способ 2: Ручной запуск

```bash
# 1. Активируйте виртуальное окружение
source venv/bin/activate

# 2. Запустите бота
python task_tracker_bot/bot.py
```

## Способ 3: Использование python3 напрямую

Если виртуальное окружение не активировано:

```bash
# Используйте python3 вместо python
python3 task_tracker_bot/bot.py
```

Но лучше активировать venv, чтобы использовать правильные зависимости.

## Проверка работы

После запуска вы должны увидеть в консоли:
```
База данных инициализирована
✅ Команда /start_test_basecase зарегистрирована
Обработчик тега @start_test_basecase зарегистрирован
Бот запущен и готов к работе!
```

## Запуск теста

После запуска бота отправьте ему в Telegram:
- `/start_test_basecase` - команда
- `@start_test_basecase` - тег в тексте
- `/test_handlers` - проверка работы обработчиков

## Остановка бота

Нажмите `Ctrl+C` в терминале для остановки бота.
